# NLP Preprocessing (Basic Python Project)

## Description
This is a simple Python program where I tried to perform basic NLP (Natural Language Processing) steps on a text.

In this program, I have taken a sentence and applied some operations like converting to lowercase, removing punctuation, splitting into words, and removing some common words (stopwords).

## What I Did
- Converted text into lowercase
- Removed punctuation marks
- Split the text into words
- Removed stopwords like "is", "a", "the", etc.

## Language Used
- Python

## How to Run
1. Install Python
2. Save the file as `nlp_preprocessing.py`
3. Run using:
   python nlp_preprocessing.py

## Output
The program shows:
- Original text
- List of words
- Filtered words after removing stopwords
